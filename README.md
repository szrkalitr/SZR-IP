# 🌍 SZR-IP

SZR-IP, tek bir IP adresi hakkında farklı kaynaklardan bilgi toplayan, hızlı ve modüler bir OSINT aracıdır. Birden fazla servisi tek komut altında birleştirerek analiz sürecini kolaylaştırır.

## 🚀 Özellikler

- IPv4 ve IPv6 desteği
- Çoklu OSINT servis entegrasyonu
- Reverse DNS sorgulama
- Risk analizi
- JSON ve CSV çıktı desteği
- Web arayüzü
- Asenkron tarama altyapısı
- Toplu IP analizi
- Modüler yapı

## 📦 Kurulum

```bash
git clone https://github.com/szrkalitr/SZR-IP.git
cd SZR-IP
unzip Szr-ip.zip
pip install -r requirements.txt
```

## ▶️ Kullanım

```bash
python szr-ip.py <IP>
```

Örnek:

```bash
python szr-ip.py 8.8.8.8
```
